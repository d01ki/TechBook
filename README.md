# TechBook 📚

技術記事を「探さなくていい」シンプルなWebアプリケーション。Zenn・Qiita・はてなブログの記事を検索キーワードで横断検索できます。

## 特徴

- 🔍 **シンプルな検索**: キーワードを入力するだけで複数メディアの記事を取得
- 🚀 **軽量設計**: ユーザー登録不要、すぐに使える
- 📱 **レスポンシブ対応**: モバイルでも快適に閲覧
- 🎯 **複数ソース対応**: Qiita、Zenn、はてなブログから記事を収集

## 使い方

### 1. インストール

```bash
npm install
```

### 2. サーバーの起動

```bash
npm start
```

### 3. アクセス

ブラウザで http://localhost:3000 を開く

### 4. 検索

検索欄にキーワードを入力（例: `JavaScript セキュリティ`）して検索ボタンをクリック

## 技術スタック

- **Backend**: Node.js + Express
- **Frontend**: Vanilla JavaScript (フレームワーク不要)
- **API統合**: 
  - Qiita API v2
  - Zenn Search API
  - Hatena Bookmark RSS

## 対応する記事ソース

- **Qiita**: APIを使用してリアルタイム検索
- **Zenn**: 公開検索APIを使用
- **はてなブログ**: はてなブックマークの人気記事から検索

## 非機能要件

以下の機能は**含まれていません**（意図的な設計）:

- ユーザー登録・ログイン
- 記事の自動学習・推薦
- 記事本文の保存・転載
- AI要約生成
